## Color Scheme

### Backgrounds
- Page background: `bg-gray-900` (#111827)
- Card background: `bg-gray-800` (#1f2937)
- Input background: `bg-gray-700` (#374151)
- Hover state: `hover:bg-gray-700` / `hover:bg-gray-600`

### Text
- Primary: `text-white`
- Secondary: `text-gray-400`
- Tertiary: `text-gray-500`
- Labels: `text-gray-300`

### Interactive Elements
- Focus ring: `focus:ring-blue-500`
- Primary action: `bg-blue-600`
- Error: `text-red-400`, `bg-red-900/20`
- Success: `text-green-500`

### Source Badges
- Riksdagen: `bg-blue-600`
- Kommun: `bg-purple-600`
- Myndighet: `bg-green-600`
- SOU: `bg-orange-600`
- Proposition: `bg-pink-600`

### Relevance Bars
- High (80-100%): `bg-green-500`
- Medium (50-79%): `bg-yellow-500`
- Low (0-49%): `bg-red-500`

## Responsive Breakpoints

- **Mobile**: < 768px
  - Full-width layout
  - Filters collapsible
  - Single column results
  - Touch-friendly buttons

- **Tablet**: 768px - 1023px
  - Search bar full-width
  - Filters still collapsible
  - Results single column
  - Optimized spacing

- **Desktop**: 1024px+
  - 4-column grid (1 filter + 3 content)
  - Filters always visible
  - Side-by-side layout
  - Full feature set

## Interactive States

### Search Input
- Default: gray border, gray placeholder text
- Focus: blue ring, blue border
- Loading: disabled state (opacity-50)
- With content: shows clear button

### Buttons
- Normal: gray border/bg
- Hover: darker gray bg
- Active/Selected: blue bg
- Disabled: opacity-50, cursor-not-allowed

### Cards
- Normal: gray border
- Hover: blue border, subtle scale
- Loading: animate-pulse

### Pagination
- Current page: blue bg
- Other pages: gray border/bg
- Hover: darker gray
- Disabled: opacity-50

## Spacing

- Container padding: `px-4 sm:px-6 lg:px-8 py-8`
- Card padding: `p-4 to p-6`
- Section spacing: `space-y-6`, `space-y-4`
- Grid gap: `gap-6`
- Element gaps: `gap-2` to `gap-4`

## Typography

- Page title: `text-3xl font-bold`
- Card title: `text-lg font-semibold`
- Section heading: `text-lg font-semibold`
- Labels: `text-sm font-medium`
- Body text: `text-sm`
- Meta text: `text-xs`

## Animation

- Transitions: `transition-all`, `transition-colors`
- Duration: 200ms (default)
- Hover effects: color/bg changes
- Page load: smooth scroll to top on pagination
- Loading state: animate-pulse on skeleton
